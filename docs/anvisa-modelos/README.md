# Modelos ANVISA – Receituários

Pasta para armazenar os modelos oficiais de receituários da ANVISA, conforme RDC 1.000/2025 e Portaria 344/98.

**Uso:** referência para layout, campos obrigatórios e validação das prescrições geradas pelo sistema.

**Fontes:**
- Portal da ANVISA: https://www.gov.br/anvisa/
- Modelos disponíveis no sítio eletrônico da Anvisa (conforme Art. 7º RDC 1.000/2025)
