# Prescrições Conforme CFM e ANVISA 2026

## Resumo das Regulamentações

### Portaria SVS/MS nº 344/98
Regulamenta medicamentos sujeitos a controle especial.

### Novos Modelos ANVISA (13/02/2026)
A partir de 13/02/2026, novos modelos de receituários entraram em vigor com campos mais detalhados.

## Tipos de Receituário

### 1. Receita Simples (Receita Branca)
- Medicamentos comuns (tarja vermelha sem retenção)
- Validade: Indefinida (recomendado 30 dias)
- Vias: 1 via
- Retenção: Não

### 2. Receita Antimicrobiana (Receita Branca 2 vias)
- Antibióticos, antifúngicos, antivirais
- Validade: 10 dias
- Vias: 2 vias (1ª via retida pela farmácia, 2ª via fica com paciente)
- Retenção: Sim
- Campos extras: Nenhum adicional

### 3. Receita B (Notificação B - Azul)
- Psicotrópicos Lista B1 e B2 (ansiolíticos, antidepressivos, anorexígenos)
- Validade: 30 dias
- Vias: 2 vias (1ª via retida pela farmácia, 2ª via fica com paciente)
- Numeração: Obrigatória da Autoridade Sanitária local
- Cor: Azul (quando impressa)

### 4. Receita A (Notificação A - Amarela)
- Entorpecentes e psicotrópicos Lista A1 e A2 (morfina, anfetaminas, etc)
- Validade: 30 dias
- Vias: 2 vias (1ª via retida pela farmácia, 2ª via fica com paciente)
- Numeração: Obrigatória da Autoridade Sanitária local
- Cor: Amarela (quando impressa)
- Limitação: Máximo 5 ampolas de substâncias injetáveis por receita

### 5. Receita C (Receita de Controle Especial - Branca 2 vias)
- Lista C1 (outras substâncias de controle especial)
- Lista C4 (Anti-retrovirais)
- Lista C5 (Anabolizantes)
- Validade: 30 dias (C1, C5) ou 6 meses (C4 - antiretrovirais)
- Vias: 2 vias (1ª via retida pela farmácia, 2ª via fica com paciente)
- Cor: Branca

## Campos Obrigatórios (CFM + ANVISA 2026)

### CABEÇALHO
1. **Logo/Nome da Instituição** (se aplicável)
2. **CNPJ/CNES da instituição** (se aplicável)
3. **Identificação do tipo** (ex: "RECEITUÁRIO DE CONTROLE ESPECIAL")
4. **Numeração sequencial** (para receitas controladas - fornecida pela Vigilância Sanitária)
5. **UF** (sigla do estado)

### IDENTIFICAÇÃO DO PRESCRITOR
1. **Nome completo** (em MAIÚSCULAS)
2. **CRM-UF** (exemplo: CRM-SP 123456)
3. **Especialidade** (opcional mas recomendado)
4. **Endereço completo** do consultório
   - Logradouro, número, complemento
   - Bairro, cidade, UF, CEP
5. **Telefone** de contato

### IDENTIFICAÇÃO DO PACIENTE (SUPERINSCRIÇÃO)
1. **Nome completo**
2. **CPF** (obrigatório desde 2026)
3. **Endereço completo**
   - Logradouro, número, complemento
   - Bairro, cidade, UF, CEP
4. **Telefone** de contato

### PRESCRIÇÃO (INSCRIÇÃO)
1. **Nome do medicamento** (DCB ou genérico preferencial)
2. **Concentração/Dosagem** (ex: 500mg, 10mg/mL)
3. **Forma farmacêutica** (comprimido, cápsula, solução, etc)
4. **Quantidade**:
   - Algarismos arábicos
   - **Por extenso entre parênteses** (OBRIGATÓRIO para controlados)
   - Exemplo: `30 (trinta) comprimidos`

### POSOLOGIA (ADSCRIÇÃO)
1. **Instruções claras e completas**
2. **Via de administração** (oral, tópica, intravenosa, etc)
3. **Frequência** (a cada 8 horas, 2x ao dia, etc)
4. **Duração do tratamento**
5. **Restrições** (não exceder X doses ao dia, tomar com alimentos, etc)

### IDENTIFICAÇÃO DO COMPRADOR (para controlados)
1. **Nome completo**
2. **CPF** ou RG (número e órgão emissor)
3. **Endereço completo**
4. **Telefone**

### IDENTIFICAÇÃO DO ESTABELECIMENTO DISPENSADOR (para controlados)
1. **Carimbo com CNPJ/CNES**
2. **Nome fantasia**
3. **Endereço completo**
4. **Data da dispensação**
5. **Assinatura e carimbo do farmacêutico** com CRF-UF

### RODAPÉ
1. **Local e data da prescrição**
2. **Assinatura do prescritor** (digital ou manuscrita)
3. **Carimbo do profissional** com nome e CRM (ou QR Code para assinatura digital)
4. **QR Code de verificação** (para receitas eletrônicas)

## Regras Específicas de Quantidade

### Receita Simples
- Quantidade livre (razoável para o tratamento)

### Receita Antimicrobiana
- Quantidade suficiente para o tratamento prescrito
- Sem limitação específica, mas deve ser razoável

### Receita B (Azul)
- Lista B1: Até 5 (cinco) ampolas por receita (se injetável)
- Outros: Quantidade conforme tratamento (geralmente 30-60 dias)

### Receita A (Amarela)
- **Máximo 5 (cinco) ampolas** de substâncias injetáveis por receita
- Comprimidos/cápsulas: Quantidade conforme tratamento (geralmente até 30 dias)

### Receita C
- C1: Quantidade conforme tratamento (geralmente 30-90 dias)
- C4 (Antiretrovirais): Quantidade para até 6 meses
- C5 (Anabolizantes): Quantidade conforme tratamento (geralmente 30 dias)

## Validade das Receitas

| Tipo | Validade | Alcance |

## Prescrição Eletrônica (2026)

### Requisitos Técnicos
1. **Assinatura Digital Qualificada** (ICP-Brasil nível 2)
2. **QR Code** com link para verificação
3. **Metadados obrigatórios**:
   - Hash SHA-256 do documento
   - Timestamp da assinatura
   - Certificado do signatário
4. **URL de verificação** pública (ex: validar.iti.gov.br)

### Plataforma CFM
- prescricaoeletronica.cfm.org.br
- Gratuita para médicos
- Integração com CFF (Conselho Federal de Farmácia)
- Validade nacional

## Observações Importantes

1. **Numeração das receitas controladas**:
   - Deve ser solicitada à Vigilância Sanitária local
   - Formato: LETRA-NÚMERO-UF (ex: B-123456789-SP)
   - Sequencial e única

2. **Duas vias obrigatórias**:
   - 1ª VIA: Retida pela farmácia (enviada à Vigilância Sanitária)
   - 2ª VIA: Orientação ao paciente

3. **Quantidade por extenso**:
   - OBRIGATÓRIA para medicamentos controlados
   - Evita adulterações
   - Deve estar entre parênteses após o número

4. **CPF obrigatório** (desde 2026):
   - Para paciente
   - Para comprador (se diferente do paciente)

5. **Receitas de outros estados**:
   - Válidas em todo território nacional (desde 2026)
   - Não precisa justificativa
   - Dentro do prazo de validade

6. **QR Code** (receitas eletrônicas):
   - Deve apontar para URL de verificação
   - Contém hash do documento
   - Permite validação pela farmácia
