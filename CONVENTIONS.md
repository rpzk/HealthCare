# Project Conventions

## General Rules

- Use TypeScript for all new code.
- Follow the existing project structure.
- Always add comments to complex logic.
- Keep components small and reusable.
- Use Tailwind CSS for styling (if already in project).

## Aider Specific

- Mention the feature or bug being worked on in the first message.
- Use `/add` to include relevant files.
- Use `/code` for specific instructions.
