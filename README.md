# HealthCare EHR: AI-Powered Clinical Management System

> **A Full-Stack Electronic Health Record (EHR) system designed by a Senior Physician to solve real-world clinical workflow inefficiencies.**

## Documentação
Toda a documentação detalhada está disponível na interface web do sistema em [docs](/docs).

## Instalação rápida
- Clone o repositório
- Configure o arquivo `.env` conforme `.env.example`
- Execute `docker compose up -d postgres redis`
- Execute `npm run dev` para ambiente local

## Stack
- Next.js 14, React, Node.js, PostgreSQL, Docker
- RBAC, AI/GenAI, integração clínica

Para dúvidas, consulte a documentação web ou abra uma issue.
